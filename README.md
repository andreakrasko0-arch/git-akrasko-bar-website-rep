# Bar Website – HTML & CSS

This is a simple website for a bar, built using HTML and CSS.  
It showcases basic website structure: header, menu, about section, contact info, and footer.

## Repository Structure
- `index.html` – main HTML file
- `style.css` – CSS stylesheet
- `images/` – images used on the website

## How to View
Open the `index.html` file in your web browser to see the website.
